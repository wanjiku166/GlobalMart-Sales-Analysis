# GlobalMart Sales Analysis

## 📖 Project Overview

This Power BI capstone project analyzes GlobalMart sales data to uncover valuable business insights through data cleaning, data modeling, Power Query, DAX measures, and interactive dashboards. The project provides a comprehensive analysis of sales performance, regional trends, product performance, and executive-level KPIs to support data-driven decision-making.

## 🎯 Project Objectives

- Analyze overall sales performance.
- Evaluate regional sales trends.
- Identify top-performing products.
- Monitor order performance.
- Provide executive-level insights for strategic decision-making.

## 🛠️ Tools & Technologies

- Power BI
- Power Query
- DAX
- Microsoft Excel
